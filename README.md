# Sales Data Analysis using Python

## Project Overview

This project analyzes sales data using Python to identify trends, top-performing products, revenue distribution, and business insights through Exploratory Data Analysis (EDA) and data visualization.

The project demonstrates practical data analysis workflows including data cleaning, feature engineering, visualization, and insight generation using Python libraries.

---

## Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

---

## Dataset Information

The dataset contains sales-related information including:

* Order Date
* Product
* Category
* Quantity
* Price
* Profit

---

## Analysis Performed

* Data Cleaning
* Handling Missing Values
* Feature Engineering
* Exploratory Data Analysis (EDA)
* Sales Trend Analysis
* Revenue Analysis
* Profit Analysis
* Data Visualization using Matplotlib

---

## Key Insights

* Electronics category generated the highest revenue
* Sales showed seasonal monthly trends
* A small number of products contributed significantly to overall sales
* Revenue and profit distribution patterns were identified through visualization
